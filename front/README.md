## React 규칙

- arrow component 사용하기
- 웬만해선 semicolon 사용하기
- **[중요]주석 잘 달기 (한국어 사용)**
- var 쓰면 사형
- 변수 명 생성 시 뜻에 맞는 단어 사용
- 함수 명은 동사를 앞으로 보내고 기본 두 단어 이상 사용. camel case 사용
- ES6 문법 사용 권장
- 작업 문서에 날짜와 본인 이름 comment
- 수정 시 수정 내용, 수정인 이름 comment
- HTML 태그는 Material User Interface lib 주로 사용하기
